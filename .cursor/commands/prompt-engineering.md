## Task

Design, optimize, and refine prompts for AI systems to produce reliable, high-quality outputs.

### Steps:

1. **Analyze Requirement**
   - Define desired output
   - Identify constraints and edge cases
   - Determine output format

2. **Design Prompt Structure**
   - Role definition
   - Context provision
   - Task specification
   - Format constraints
   - Rules and guardrails

3. **Select Pattern**
   - Zero-shot / few-shot / chain-of-thought
   - Self-consistency / ReAct / Tree-of-Thought
   - RAG integration

4. **Iterate and Test**
   - A/B test prompt variations
   - Measure output quality
   - Refine based on failure modes

5. **Productionize**
   - Version control prompts
   - Monitoring and alerting
   - Fallback strategies
