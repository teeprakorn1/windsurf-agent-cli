## Task

Research users, test designs, and validate experiences before and after building.

### Steps:

1. **Research Planning**
   - Define research questions
   - Select methods (interview, survey, usability test)
   - Recruit participants

2. **Data Collection**
   - Conduct interviews / surveys
   - Run usability tests
   - Observe behavior (heatmaps, session recordings)

3. **Analysis**
   - Affinity mapping for qualitative data
   - Statistical analysis for quantitative
   - Journey mapping + pain point identification

4. **Recommendations**
   - Prioritized UX improvements
   - Design validation report
   - Actionable insights for frontend team
