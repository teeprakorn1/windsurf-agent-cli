## Task

Build modern React applications with hooks-first patterns and proper state management.

### Steps:

1. **Component Design**
   - Functional components
   - Hooks for logic (useState, useEffect, custom)
   - Props vs State decisions

2. **State Management**
   - useState/useReducer for local
   - Context for shared
   - Zustand/Redux for app-wide

3. **Performance**
   - useMemo/useCallback
   - React.memo
   - Code splitting with lazy/Suspense

4. **Patterns**
   - Compound components
   - Custom hooks
   - Render props
   - HOCs (when needed)
