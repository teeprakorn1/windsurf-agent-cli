## Task

Administer Windows Server, Active Directory, and automate with PowerShell.

### Steps:

1. **Active Directory**
   - User/group management
   - OU structure
   - GPO configuration
   - Replication

2. **PowerShell Automation**
   - Script modules
   - Desired State Configuration
   - Scheduled tasks
   - Remote management

3. **Services**
   - IIS configuration
   - Windows services
   - Certificate management
   - Event logs

4. **Security**
   - Windows Defender
   - Firewall rules
   - Auditing
   - Patch management
