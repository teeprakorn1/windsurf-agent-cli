## Task

Build semantic, performant web experiences without JavaScript frameworks.

### Steps:

1. **Semantic HTML**
   - Proper document structure
   - Accessibility landmarks
   - SEO-friendly markup

2. **Modern CSS**
   - CSS custom properties
   - Grid + Flexbox layouts
   - Container queries
   - Logical properties

3. **Vanilla JS (when needed)**
   - Modern selectors
   - Event delegation
   - Intersection Observer
   - Fetch API

4. **Performance**
   - Critical CSS inline
   - Lazy loading
   - Font optimization
   - No render-blocking
