## Task

Prepare application for multiple languages and regions. Detect hardcoded strings, set up translation infrastructure, and ensure locale-aware formatting.

### Steps:

1. **Audit Current State**
   - Detect hardcoded strings
   - Check date/number/currency formatting
   - Test RTL readiness

2. **i18n Architecture**
   - Set up translation file structure
   - Configure i18n library (react-i18next, next-intl, etc.)
   - Implement locale detection + fallback chain

3. **String Extraction**
   - Move all text to translation files
   - Use ICU MessageFormat for plurals + interpolation
   - Add context to keys (not `btn1`, use `user.profile.save_button`)

4. **Layout Adaptation**
   - RTL support (Arabic, Hebrew)
   - Text expansion buffer (German +30%)
   - Flexible containers (no fixed-width text)

5. **Verification**
   - Pseudolocalization test (accented chars)
   - RTL layout test
   - Date/number/currency format test per locale
