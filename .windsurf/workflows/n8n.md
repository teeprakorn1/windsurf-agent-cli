## Task

Build n8n workflows to automate business processes and system integrations.

### Steps:

1. **Trigger Selection**
   - Webhook, schedule, or event-based
   - Input validation

2. **Data Flow Design**
   - Node connections
   - Data transformation
   - Error handling

3. **Integration**
   - API connections
   - Database operations
   - Third-party services

4. **Logic**
   - Conditional branching
   - Loops and iteration
   - Wait states

5. **Monitoring**
   - Error workflows
   - Execution logging
   - Notifications
